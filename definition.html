Instalation en local

1- Téléchargr le derniere version
2- Extraire le dossier dans Htdocs
3- Se rendre sur l'adresse localhost avec le navigateur
4- Cliquer sur le bouton "C'est partis !"
5- Se rendre dans PhpMyAdmin pour créer la BDD
6- Revenir sur l'onglet pour renseigner tous les champs relatif à la BDD
7- Lancer l'instalation.
8- Une fois xampp lancé on va sur http://localhost/wordpress
9- Aller dans le dossier htdoc ouvrir le index.php avec vscode et modifier le header loacation dashbord devient wordpress // ou supprimer l'index
mettre root pour l'identifiant
